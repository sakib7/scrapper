# Scrapper

### Script to collect GSOC 2017 org links and the projects

#### List of used technologies:

* Python 3.6
* Beautiful Soup 4
* Selenium Web Driver
* xlrt + xlrt

#### List of development tools:

* pycharm-community-2017.1.exe
* Anaconda3-4.3.1-Windows-x86_64.exe
* Chromedriver.exe
